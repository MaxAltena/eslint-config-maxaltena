## Installation

Install the package with

```sh
npm install @maxaltena/eslint-config-maxaltena --save-dev
# or
yarn add @maxaltena/eslint-config-maxaltena -D
```

## Usage

Now add the config to either your `package.json`:

```json
{
	"eslintConfig": {
		"extends": "eslint-config-maxaltena"
	}
}
```

to your `.eslintrc`:

```json
{
	"extends": "eslint-config-maxaltena"
}
```

or `.eslintrc.js`:

```js
module.exports = {
	extends: "eslint-config-maxaltena",
};
```
